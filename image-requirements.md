# TrustIn Website - Image & Asset Requirements

## Overview
This document outlines all images and visual assets needed for the TrustIn healthcare licensing lead generation website.

---

## Image Specifications

### General Requirements
- **Format**: WebP (primary), PNG/JPG (fallback)
- **Color Profile**: sRGB
- **Optimization**: Compressed for web (max 200KB per image)
- **Naming Convention**: descriptive-kebab-case.webp
- **Alt Text**: Required for all images (documented below)

## Accessibility Notes

- All images must have descriptive alt text
- Decorative images should have empty alt="" attribute
- Icons used for meaning should have aria-labels
- Ensure sufficient color contrast in images with text
- Provide text alternatives for complex infographics
